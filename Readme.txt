An interactive online quiz app built using HTML, CSS, and JavaScript.  
Supports category selection, difficulty levels, timers, dark mode, and detailed answer reviews!

## Features
- 10 Random Questions from Open Trivia API
- Choose Category and Difficulty (Easy/Medium/Hard)
- Countdown Timer for each question
- Dark/Light Mode Toggle
- Answer Summary at the end
- Responsive Design (Mobile-Friendly)

## Demo

[Live Demo Link (Optional if you deploy)](https://your-demo-link.com)

## How to Use
1. Select category and difficulty.
2. Answer the questions within the given time.
3. View your results and answers at the end.
4. Restart and try again!

## Tech Stack
- HTML5
- CSS3 (Flexbox, Responsive Design)
- JavaScript (Fetch API, Dynamic Rendering)
